Base De Datos ETN-1000

La materia de Base de datos es fundamental en la formacion de ingenieros en la carrera de ingenieria Electronica. Estos titulados seran quienes gestion y exploten los sistemas informaticos de empresas y organizaciones, siendo la base de datos una parte clave de Estos sistemas. Cada vez mas, los sistemas de gestion de base de datos proporcionan herramientas que dan soporte a todo el ciclo de vida de los sistemas informacion, por lo que su estudio resulta imprescindible.
